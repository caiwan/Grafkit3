### Prerequisites
1. Install [Vulkan SDK](https://vulkan.lunarg.com/sdk/home)
2. Install [CMake](https://cmake.org/download/)
3. Install Clang
4. Install python3

### Initialize subrepositories

```bash
git submodule update --init --recursive
```
